# Search Bar

A simple and clean search bar component built using HTML and CSS. It demonstrates how to design and style a functional search input for web interfaces.

## Features
- Minimal and responsive design  
- Smooth and modern styling  
- Works in all modern browsers  
- No JavaScript required (UI only)

## How It Works
The project focuses on layout and styling. The search bar is built with semantic HTML and styled using CSS to create a neat, centered input field that can be reused in any project.
